# LinkedIn Reality Check & Jargonizer 🔄💎

The ultimate "Secret Decoder Ring" for corporate life. Unmask the cringe in your feed, or recode your truths into professional prestige.

![Preview Image Card Placeholder]

## 🌟 The Mission
LinkedIn has become a soup of corporate buzzwords and vague announcements. Our mission is two-fold:
1. **Reality Check**: Paste any corporate post to see what's *actually* being said. 🔎
2. **Jargonize**: Paste your simple truths and achievements to translate them into professional, jargon-packed prestige assets. ✨

## 🚀 Features
*   **Dual-Mission Console**: Switch between "Reality Check" and "Jargonizer" based on your needs.
*   **AI-Driven Decoding**: Uses advanced language models to identify buzzwords and corporate-speak.
*   **Visual Share Cards**: Generates beautiful, portrait-ratio (4:5) image cards optimized for LinkedIn, Instagram, and more.
*   **Native Image Sharing**: Direct integration with your mobile device's share sheet for instant viral potential. 🤳
*   **Live Social Proof**: Sticky banner showing global usage (Checks Run & Live Visitors). 📈
*   **One-Tap Inspiration**: Context-aware example chips to test the tool instantly. 💡

## 🛠️ Tech Stack
*   **Frontend**: React + Vite
*   **Styling**: Vanilla CSS (Tailwind Utilities)
*   **Animations**: Framer Motion
*   **Icons**: Lucide React
*   **Image Generation**: HTML-to-Image
*   **Stats API**: CounterAPI.dev

## 📦 Getting Started
1. **Clone & Install**:
   ```bash
   npm install
   ```
2. **Run Locally**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```
